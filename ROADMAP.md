# Roadmap

This roadmap describes the planned evolution of HEYNA REPORT.

## v1.0

Current features:

- Playwright integration
- Page Object Model support
- Screenshot Evidence per step
- API Logging
- Execution Summary
- Test Case Summary
- Step Summary
- Failed Test Analysis
- PDF Report Generator
- Custom Branding
- Enterprise-style layout

## v2.0

Planned features:

- Pie Chart for test status summary
- Execution Trend summary
- Enhanced Footer customization
- Config file for branding and metadata
- Better failed test grouping

## v3.0

Planned features:

- Video Evidence
- HAR Capture
- Trace Viewer links
- Network timeline summary
- Attachment indexing

## v4.0

Planned features:

- HTML Dashboard
- Multi Project Support
- Theme Support
- Dark and light report themes
- Filterable test result views

## v5.0

Planned features:

- NPM Package
- CI/CD Integration templates
- CLI command for report generation
- Reusable project scaffolding command
- Report publishing adapters
